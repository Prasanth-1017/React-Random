# Random Number Generator Web App

An interactive random number generator application built with **React** and **Vite**, featuring a clean, responsive design powered by **Tailwind CSS**.

## Features

- **Random Number Generator**: A button that generates a random number between 1 and 100 each time it's clicked.
- **User Personalization**: A welcoming interface that allows users to set a display name.
- **Visual Feedback**: A message that appears when the random number is generated, fading in and out within 1500ms.
- **Responsive UI**: Optimized for various screen sizes with a modern, minimalistic aesthetic.

## Getting Started

Follow these steps to run the application locally:

1. **Install Dependencies**

    ```bash
    npm install
    ```

2. **Run Development Server**

    ```bash
    npm run dev
    ```
